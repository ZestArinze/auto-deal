# Auto Deals - Car Rentals API

An Node js API for car rentals... for demonstration purposes. Build with love with Nest js.

## Installation

```bash
$ git clone https://github.com/ZestArinze/auto-deal.git
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
